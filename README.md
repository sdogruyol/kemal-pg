# kemal-pg

Postgresql connection middleware for [Kemal](https://github.com/sdogruyol/kemal)

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  kemal-pg:
    github: sdogruyol/kemal-pg
```


## Usage


```crystal
require "kemal-pg"
```
